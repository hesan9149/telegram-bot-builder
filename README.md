# Telegram Bot Builder

این پروژه یک ربات‌ساز تلگرام با پایتون است.

## امکانات

- ۳۰۰ قالب ربات
- سیستم امتیاز
- سیستم رفرال
- دریافت ۲ امتیاز برای هر زیرمجموعه
- جوین اجباری در کانال:
  https://t.me/Hack_R4ei30
- کسر امتیاز برای ساخت هر نوع ربات
- اتصال به پشتیبانی برای خرید امتیاز

## نصب

```bash
pip install -r requirements.txt
```

## اجرا

اول داخل فایل `bot.py` مقدار زیر را تغییر بده:

```python
BOT_TOKEN = "توکن_ربات_خودت_را_اینجا_بگذار"
```

بعد اجرا کن:

```bash
python bot.py
```

## نکته مهم

رباتی که با BotFather می‌سازی باید داخل کانال جوین اجباری ادمین باشد، وگرنه نمی‌تواند عضویت کاربران را بررسی کند.
